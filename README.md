📇 Smart Profile Card with AI-Driven Design
A sleek, interactive, and fully responsive Profile Card built using Bootstrap 5 and vanilla JavaScript. This project focuses on modern UI/UX principles and AI-assisted code optimization.

🚀 Project Overview
This is not just a static card; it's a digital identity widget designed to be integrated into portfolios or dashboards. It features smooth hover animations, social media integration, and a dynamic layout that works on any device.

🛠 Tech Stack
HTML5: Semantic structure for better SEO and accessibility.

CSS3: Custom styling and smooth transition animations.

Bootstrap 5: For a mobile-first responsive grid system.

JavaScript: To handle interactive elements and dynamic data.

AI Tools: Refactored using DeepSeek and Blackbox AI for efficient CSS architecture.

✨ Key Features
Responsive Design: Looks perfect on mobile, tablet, and desktop.

Glassmorphism Effects: Modern UI style with subtle blurs and shadows.

Social Link Integration: Interactive buttons for LinkedIn, GitHub, and Twitter.

Hover Interactivity: Animated state changes to enhance user engagement.

AI-Optimized: Clean, documented code structure following industry best practices.
